# Cycle Layout Template

Purpose: Optional folder layout for parallel ARCH-AEP cycles.

## Example Layout
docs/ARCH AGENTIC ENGINEERING AND PLANNING/cycles/YYYY-MM-DD/
- backlog-YYYY-MM-DD.md
- scope-lock-YYYY-MM-DD.md
- tracker-YYYY-MM-DD.md
- phase-summaries/
- risk-memos/
- verification-log.md (default for evidence logs)
- change-log.md
- agent-learning.md

Note: Add entries to `backlog-index.md` and `tracker-index.md` using the cycle folder paths.
